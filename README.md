# PixelPlay
> A basic photo editor webapp using HTML, CSS, JS, Java

## Tech Stack
| Front-End | Back-End  | Database   |
| --------  | --------- | ---------- |
| HTML      | Java      | MySQL      |
| CSS       | Firebase  | &nbsp;     |
| JS        | &nbsp;    | &nbsp;     |

## *Features*
 - Crop
 - Rotate
 - Brightness
 - Contrast
 - Saturation
 - Opacity
 - Temperature
 - Blur
 - Preset 
 - Undo edit
 - Redo edit
 - Download image
